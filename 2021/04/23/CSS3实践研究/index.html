3. 布局–从block开始

3.1 block-level element

元素 display
<div> block
<li> list-item
<table> block

一个水平流上只能单独显示一个元素，多个块级元素则换行显示。
正是由于块级元素具有换行特性，因此理论上它都可以配合clear属性
来清除浮动带来的影响。

简化layout解释

a. 不考虑list-item的情况下
| 盒子 | 职责 |
| - | - |
| block-level box | 结构 |
| inline box | 内容 |

b. 考虑list-item当前情况下

盒子 职责

inline box 内容
